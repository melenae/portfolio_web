# Портфолио сайт

Современный сайт-портфолио на React.js с красивым дизайном и адаптивной версткой.

## Структура проекта

- **Header** - Навигационная панель с меню
- **Home** - Главная страница с секциями:
  - Обо мне
  - Проекты
  - Обратная связь
- **ProjectDetail** - Страница отдельного проекта

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект в режиме разработки:
```bash
npm start
```

Проект откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)

3. Для сборки продакшн версии:
```bash
npm run build
```

## Настройка

### Изменение данных проектов

Отредактируйте файл `src/data/projects.js` и добавьте свои проекты:

```javascript
{
  id: 1,
  title: 'Название проекта',
  description: 'Краткое описание',
  fullDescription: 'Подробное описание проекта',
  image: 'путь/к/изображению.jpg', // или URL
  bgColor: '#667eea', // Цвет фона страницы проекта
  technologies: ['React', 'Node.js'], // Используемые технологии
}
```

### Изменение информации о себе

Отредактируйте файл `src/components/About.js` и добавьте свою информацию.

### Настройка контактной информации

Отредактируйте файл `src/components/Contact.js` и измените:
- Email
- Телефон
- Добавьте логику отправки формы (например, через EmailJS или API)

### Изменение имени в навигации

Отредактируйте файл `src/components/Header.js` и замените "Ваше Имя" на свое имя.

### Настройка ссылки на Instagram

В файле `src/components/Header.js` замените `https://instagram.com` на ссылку на ваш Instagram профиль.
